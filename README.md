# Billiard-Hall

Full-stack billiard hall website with React frontend and Express backend for reservations and signups.

## Environment variables

The backend now requires MongoDB.

Set these variables locally and in Vercel:

- `MONGODB_URI`
- `MONGODB_DB_NAME`
- `ADMIN_PASSWORD`

Example values are provided in [.env.example](c:/Users/giorg_11uniim/Desktop/billiard/.env.example).
